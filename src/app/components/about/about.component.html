Prueba Kushki

Esta página simula una tienda online de cursos para probar la funcionalidades Request a card token y make a charge tal como indica el instructivo de la prueba de Kushky.

Para este proyecto se usó el framework angular en su versión 12.1.2 en el backend se utilizó node con su versión 6.14.8 y para levantar el servidor el framework express en su version 4.17.1

Pare el diseño web se uso boostrap en su versión 4, además se manejó angular material para desarrollar el formulario para la pasarela de pagos.

La página web se encuentra alojada en firebase hosting y el backend se desplegado en heroku.
